Convert ACSM files to DRM-free EPUB files

This web interface provides a user-friendly way to use Knock, a tool that converts ACSM files to DRM-free EPUB files using one command.

Note: This web app serves as a guide and interface. The actual conversion requires the Knock CLI tool or a compatible backend service.

CLI Installation (Alternative)

For direct CLI usage, install Knock on your system:

# Add to your system flake.nix
inputs.knock.url = github:BentonEdmondson/knock
outputs = { self, knock }: { /* knock.defaultPackage.x86_64-linux is the package */ }

# Then run:
knock ~/path/to/my-book.acsm
# Download the latest release
# Make it executable
chmod +x knock-version-x86_64-linux
mv knock-version-x86_64-linux knock

# Run conversion
./knock ~/path/to/my-book.acsm
